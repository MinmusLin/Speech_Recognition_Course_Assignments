Speech_Recognition_Course_Assignments

## 仓库简介

A collection of my speech recognition course assignments.

2024年同济大学语音识别课程作业合集。

> ***Relevant course***
> * Speech Recognition 2024 (2024年同济大学语音识别)

* [MindSpore DeepSpeech2](MindSpore_DeepSpeech2)
